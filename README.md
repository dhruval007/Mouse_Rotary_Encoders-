<h3>ğŸ§© Rotary Encoder Internal Structure â€“ Brief Overview</h3>
<p>The scroll wheel uses a mechanical rotary encoder built from a few simple components.  
The <b>steel casing</b> provides structural support and holds everything in place, while the internal <b>clips</b> act as spring contacts that rub against the <b>turntable</b>. The turntable contains conductive segments that rotate as the wheel moves, generating pulse signals.  
These signals pass through the <b>plastic base with three output pins</b>, allowing the mouse MCU to detect scroll direction and speed.</p>

<h1>Pulse generated by the mouse scroll wheel.</h1>

<img width="597" height="293" alt="image" src="https://github.com/user-attachments/assets/3699e8e1-9d67-4497-b8ba-fbac4f5812ae" />

<p>The rotary encoder generates digital pulse signals as the wheel rotates. Each notch produces a high-low transition forming a square-wave output. When you scroll faster, the pulses occur more frequently and the width becomes narrower. The microcontroller counts these pulses to detect scroll movement and speed.</p>

<h1>Photosensitive Rotation</h1>

<img width="875" height="491" alt="image" src="https://github.com/user-attachments/assets/896a57c3-7ceb-450c-99b1-60d80ae3db79" />

<p>This type of scroll wheel uses a photosensitive sensor instead of a mechanical encoder. 
An LED shines light through the slotted wheel, and a photodiode detects the interruptions 
as the wheel rotates. Each slot passing the beam generates pulses which the controller 
interprets as scroll movement. When replacing the sensor or wheel, the installation height 
and alignment must match the original for accurate detection.</p>
